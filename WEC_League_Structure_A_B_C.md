# WEC League Structure: A/B/C Tier System
## World Espresso Championship Competitive Pyramid

**Document Date:** 2026  
**Purpose:** Define the three-tier league structure for WEC competitive ecosystem

---

## Executive Overview

The WEC competitive structure operates as a **three-tier pyramid** with distinct connection points between levels:

```
                    WORLD ESPRESSO CHAMPIONSHIP
                           (PREMIER)
                         A LEAGUE — TIER 1
                               ▲
                               │ CONNECTED
                               │
                    NATIONAL CHAMPIONSHIPS
                               ▲
                               │ CONNECTED
                               │
                       RURAL/REGIONAL EVENTS
                               ▲
                               │
       ┌───────────────────────┼───────────────────────┐
       │                       │                       │
       ▼                       ▼                       ▼
   B LEAGUE —          ESPRESSO THROWDOWNS —      [OPEN]
   TIER 2              TIER 3/ENTRY LEVEL
   (Practice)          (Community)
   NOT CONNECTED       NOT CONNECTED
```

**Key Principle:** Only the A League has upward mobility. B League and Throwdowns are parallel tracks for practice and community — they don't feed into the Premier League directly.

---

## TIER 1: A LEAGUE — THE PREMIER LEAGUE

### Structure (Connected System)

```
RURAL EVENT → NATIONAL CHAMPIONSHIP → WORLD ESPRESSO CHAMPIONSHIP
   (Local)         (Country)                (Global)
     ▲                ▲                        ▲
     │                │                        │
   Open            Winners of              Winners of
   Qualifying      Rural Events            National
                   from that               Championships
                   country                 from each
                                           country
```

### Movement Rules

| Level | How You Get In | What Happens If You Win |
|-------|----------------|------------------------|
| **Rural/Regional** | Open registration OR qualifier results | Invitation to National Championship |
| **National Championship** | Win Rural event in that country | Invitation to World Espresso Championship |
| **World Championship** | Win National Championship | Premier League Champion title + prize pool |

### Sponsor Model: Permanent Partnership

**A League sponsors are LOCKED IN:**
- Official equipment sponsors (Dalla Corte, etc.)
- Official grinder sponsors
- Official coffee partners
- These sponsors are tied to **ALL** A League events forever
- Cannot swap sponsors for individual events

**Why:** Brand consistency, long-term partnerships, sponsor investment security

### Event Characteristics

| Aspect | Specification |
|--------|---------------|
| **Equipment** | Standardized (all competitors use same) |
| **Format** | Official WEC rules (A vs B, binary preference) |
| **Judging** | Certified WEC judges only |
| **Prize Pool** | Funded by Premier League sponsors + membership revenue |
| **Media** | Full production, live streaming, documentary coverage |
| **Data** | Full competitive data collected (profiles, scores, preferences) |

## TIER 2: B LEAGUE — PRACTICE LEAGUE

### Structure (NOT Connected)

```
B LEAGUE EVENTS
├── Single Event #1 (Sponsor: Local Roaster A)
├── Single Event #2 (Sponsor: Equipment Company B)
├── Single Event #3 (Sponsor: Café Chain C)
└── etc.

NO PATH TO A LEAGUE — These are practice/competitions for their own sake
```

### Purpose

- **Practice ground** for A League competitors
- **Development** for aspiring competitors
- **Flexibility** for experimentation (formats, rules, equipment)
- **Revenue** from individual event sponsorships

### Movement Rules

| Direction | Possible? | Notes |
|-----------|-----------|-------|
| B League → A League | ❌ NO | Winning a B League event does NOT qualify you for Nationals |
| A League → B League | ✅ YES | A League competitors can enter B League for practice |
| B League → B League | ✅ YES | Open to all; no qualification needed |

**Key Point:** B League is a **parallel track**, not a ladder. It exists for practice, experimentation, and single-event partnerships.

### Sponsor Model: Single Event, Flexible

**B League sponsors are EVENT-SPECIFIC:**
- Each B League event can have **different sponsors**
- No permanent ties to A League sponsors
- Allows WEC to work with:
  - Local roasters
  - Regional equipment dealers
  - Café chains
  - One-off partnerships

**Example:**
- B League Event 1: Sponsored by Local Roaster X
- B League Event 2: Sponsored by Grinder Company Y (not the A League grinder sponsor)
- B League Event 3: Sponsored by Café Chain Z

**Strategic Value:** 
- Test new sponsor relationships
- Local/regional revenue opportunities
- No conflict with Premier League partners

### Event Characteristics

| Aspect | Specification |
|--------|---------------|
| **Equipment** | Variable (can be sponsor-provided OR competitor-owned) |
| **Format** | Flexible (can mirror A League or experiment) |
| **Judging** | Can use local judges, simplified rules, or guest judges |
| **Prize Pool** | Set by event sponsor/organizer |
| **Media** | Optional (can be simple phone video or full production) |
| **Data** | WEC collects competitive data (but not tied to A League rankings) |

## TIER 3: ESPRESSO THROWDOWNS — COMMUNITY LEAGUE

### Structure (NOT Connected / Entry Level)

```
ESPRESSO THROWDOWNS
├── Joe's Café Throwdown (Seattle)
├── Downtown Community Event (Melbourne)
├── College Coffee Club Comp (London)
├── Barista Meetup Throwdown (Tokyo)
└── [Anyone can run one]

ZERO SPONSORSHIP REQUIRED
ZERO WEC ORGANIZATION REQUIRED
WEC COLLECTS THE DATA
```

### Purpose

- **Entry point** for new competitors
- **Community building** at local level
- **Zero barrier** to entry (anyone can run one)
- **Data collection** for WEC ecosystem
- **Organic growth** — no WEC resources required

### How It Works

**For Hosts:**
1. Have a coffee machine, grinder, and coffee
2. Set a date, invite people
3. Run the throwdown (informal format)
4. Submit results to WEC

**For WEC:**
1. Provide basic guidelines (optional)
2. Collect data from throwdowns
3. Track competitor development
4. Identify emerging talent
5. Feed community stories to content pipeline

### Movement Rules

| Direction | Possible? | Notes |
|-----------|-----------|-------|
| Throwdown → B League | ❌ NO | No formal connection |
| Throwdown → A League | ❌ NO | No formal connection |
| Throwdown → Throwdown | ✅ YES | Open participation |
| Anyone can host | ✅ YES | Self-organized |

**Key Point:** Throwdowns are **completely open**. No qualification, no pathway, no official status. They're the grassroots — WEC just harvests the data.

### Sponsor Model: None (Or Optional)

**No WEC-mandated sponsorship:**
- Hosts can fund themselves
- Hosts can find local sponsors if they want
- WEC provides zero financial support
- WEC provides zero equipment

**Optional WEC Support:**
- Digital toolkit (how to run a throwdown)
- Branded materials (posters, etc.)
- Data collection form/app
- Recognition on WEC platform

### Data Collection (The Strategic Advantage)

**What WEC Collects:**
- Competitor names and profiles (emerging talent)
- Competition results
- Geographic distribution of activity
- Equipment used
- Coffee preferences
- Audience sizes

**Why This Matters:**
- Identifies talent before they reach A League
- Maps the global coffee competition landscape
- Feeds content pipeline ("community stories")
- Builds database of engaged coffee enthusiasts
- Future recruitment for B League and A League

### Event Characteristics

| Aspect | Specification |
|--------|---------------|
| **Equipment** | Host provides (whatever they have) |
| **Format** | Host decides (simplified WEC format recommended) |
| **Judging** | Peer judging, audience voting, or guest judges |
| **Prize** | Optional (bragging rights, small prizes, coffee bags) |
| **Media** | Host-generated (phone videos, social posts) |
| **Data** | WEC collects via submission form |

| Feature | A LEAGUE (Premier) | B LEAGUE (Practice) | THROWDOWNS (Community) |

## COMPETITOR JOURNEY EXAMPLES

### Journey 1: The Organic Rise

```
ESPRESSO THROWDOWN (Local café)
    ↓ [WEC identifies talent via data]
B LEAGUE EVENT (Practice competition)
    ↓ [Competitor decides to pursue serious competition]
RURAL EVENT (A League qualifying)
    ↓ [Wins]
NATIONAL CHAMPIONSHIP
    ↓ [Wins]
WORLD ESPRESSO CHAMPIONSHIP
```

**Note:** The Throwdown and B League didn't "qualify" the competitor — they were practice/visibility. The competitor still had to enter and win the Rural event to start the A League path.

### Journey 2: The Direct Entry

```
RURAL EVENT (Registers directly)
    ↓ [Wins]
NATIONAL CHAMPIONSHIP
    ↓ [Wins]
WORLD ESPRESSO CHAMPIONSHIP
```

**Note:** Skipped Throwdowns and B League entirely. Valid path — A League is open to all who qualify through the connected system.

### Journey 3: The B League Regular

```
B LEAGUE EVENT #1 (Practice)
B LEAGUE EVENT #2 (Practice)
B LEAGUE EVENT #3 (Practice)
    ↓ [Never enters A League — stays in B League for fun/practice]
[Continues competing in B League events]
```

**Note:** Some competitors will prefer the flexibility of B League. They never have to enter the A League system.

## STRATEGIC IMPLICATIONS

### For WEC

| League | Strategic Value |
|--------|-----------------|
| **A League** | Brand prestige, sponsor revenue, media content, "main product" |
| **B League** | Sponsor flexibility, format experimentation, practice ecosystem |
| **Throwdowns** | Data collection, talent identification, community growth, zero cost |

### For Competitors

| Goal | Recommended Path |
|------|------------------|
| Win World Championship | A League only (Rural → National → World) |
| Practice and improve | B League + occasional A League |
| Have fun, community | Throwdowns + B League |
| Try competition without commitment | Throwdowns |

### For Sponsors

| Sponsor Type | Best Fit |
|--------------|----------|
| Global equipment manufacturers | A League (permanent partnership) |
| Regional roasters | B League (single events) |
| Local cafés | Throwdowns (host their own) |

## SUMMARY

The WEC three-tier system creates a **complete competitive ecosystem:**

1. **A LEAGUE (Premier):** The path to glory. Connected, standardized, sponsor-locked.
2. **B LEAGUE (Practice):** The training ground. Flexible, experimental, single-sponsor.
3. **THROWDOWNS (Community):** The entry point. Open, self-organized, data-generating.

**The genius:** Throwdowns cost WEC nothing but generate massive data. B League generates revenue from flexible sponsorships. A League generates prestige and permanent sponsor value.

**The connection rule:** Only A League feeds A League. Everything else is parallel.
